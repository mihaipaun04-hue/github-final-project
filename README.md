# Simple Interest Calculator

## Description
A simple interest calculator that computes the interest earned on a principal amount over a period of time at a given rate.

## Formula
Simple Interest = (Principal * Rate * Time) / 100

## Inputs
- **Principal**: The initial amount of money
- **Rate**: The annual interest rate (in percentage)
- **Time**: The time period (in years)

## Output
- **Simple Interest**: The interest amount calculated

## Example
- Principal = 1000
- Rate = 5
- Time = 3
- Simple Interest = (1000 * 5 * 3) / 100 = **150**
